== Description ==

The official Unlimit plugin allows you to process payments for your online store built on WooCommerce engine, allowing users to finalize their purchase with their preferred payment method. Ensure compatibility with MySQL version 5.6 or greater OR MariaDB version 10.1 or greater as per the WooCommerce server requirements.
